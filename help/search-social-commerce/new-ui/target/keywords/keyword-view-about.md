# (New UI) About the [!UICONTROL Keywords] view

*Beta feature*

The [!UICONTROL Manage] > [!UICONTROL Keywords] view includes two tabs:

* The [!UICONTROL Keywords] tab lists all keywords in the filtered view for the selected advertiser account.

* The [!UICONTROL Negatives] tab lists all negative keywords in the filtered view for the selected advertiser account.
