# (New UI) About the [!UICONTROL Placements] view

*Beta feature*

The [!UICONTROL Manage] > [!UICONTROL Placements] view includes two tabs:

* The [!UICONTROL Placements] tab lists all placements in the filtered view for the selected advertiser account.

* The [!UICONTROL Negatives] tab lists all negative placements in the filtered view for the selected advertiser account.
