# View simulation details

<!-- Needs subfeature tag "portfolios -->

*Beta feature*

You can view simulation details on-screen as a graph or table.

1. In the main menu, click **[!UICONTROL Plan] > [!UICONTROL Simulations]**.

1. Click the simulations name.

   The simulation opens in the graph view.

1. (Optional) In the graph view, do any of the following:

   * To change the metric for which estimated data is shown for each spend target level (step), select a new metric from the **[!UICONTROL Y-Axis Metric]** menu.

   * (Weekly simulations only) Enter a new **[!UICONTROL Target Mid Point]** value to see the predicted data using that value.

   * (Weekly simulations only) From the **[!UICONTROL Constraints]** menu, choose whether to see data *[!UICONTROL With Constraints]*, *[!UICONTROL Without Constraints]*, or *[!UICONTROL With and Without]*.

1. (Optional) To open the table view, click the **[!UICONTROL Table View]** tab.

1. (Optional) To see the read-only settings for the applicable portfolio, click **[!UICONTROL Portfolio Settings]** in the upper right.
