# (New UI) View a portfolio's change history

*Beta feature*

You can view a log of changes made to the portfolio during any date range. The actions logged include the portfolio creation, all changes to the portfolio's settings, all campaigns assigned to or unassigned from the portfolio, all constraints applied to or removed from bid units in the portfolio, all constraints on bid units in the portfolio that have expired, and other actions that affect the portfolio

You can filter the data by category<!-- Not available as of 5-14: and by the user who performed the action -->. You can also sort the list by date in ascending or descending order.

You can also download the data to a spreadsheet file.

## (New UI) View a portfolio's change history {#portfolio-change-history}

Information for each change includes the date, the user name of the person who made the change, the change type, and the changed value.

1. In the main menu, click **[!UICONTROL Goals] > [!UICONTROL Conversions]**.

1. Hold the cursor over the portfolio row and click **[!UICONTROL ...] > [!UICONTROL Change History]**.

1. (Optional) Change the information that's displayed:

   * Click ![Calendar](/help/search-social-commerce/assets/calendar-new.png "Calendar") to change the date range for the report.

   * Click ![Filter](/help/search-social-commerce/assets/filter-new.png "Filter") to filter the data by change type.<!-- Not available as of 5-14: and by the user who performed the action -->.

   * Click ![Sort](/help/search-social-commerce/assets/sort.png "Sort") to sort the list by date in ascending or descending order.


## Manage performance data reports for change histories

You can download additional information about the change history for a portfolio to a file in [!DNL Microsoft Excel] workbook (XLSX file) format. The report includes the numeric change ID, the old and new values, the change type (such as "added" or "modified"), the action type (such as "Budget" or "Portfolio Name") and subtype when applicable, the time, and the user name.

### Generate a report with the filtered data rows

1. [Open the portfolio's change history](#portfolio-change-history).

1. In the upper right, click ![Download Report](/help/search-social-commerce/assets/download.png "Download Report").

1. In the [!UICONTROL Grid Reports] settings, enter a unique report name, and then click **[!UICONTROL Generate]**.

   By default, the file is named "portfolioChangeHistory_YYYYMMDD_NNNN," where "NNNN" is the sequential job number (such as "portfolioChangeHistory_20260402_1326).

   The file is added to the [!UICONTROL Recently Generated] list.

1. (Optional) To download the file once it's completed, click ![Download](/help/search-social-commerce/assets/download.png "Download") next to the file name.

   The file is downloaded according to your browser's normal procedure.

### Download a completed report

1. [Open the portfolio's change history](#portfolio-change-history).

1. In the upper right, click ![Download Report](/help/search-social-commerce/assets/download.png "Download Report").

1. In the [!UICONTROL Recently Generated] list in the [!UICONTROL Grid Reports] dialog, click ![Download](/help/search-social-commerce/assets/download.png "Download") next to the file name.

   The file is downloaded according to your browser's normal procedure.

### Delete a completed report

1. [Open the portfolio's change history](#portfolio-change-history).

1. In the upper right, click ![Download Report](/help/search-social-commerce/assets/download.png "Download Report").

1. In the [!UICONTROL Recently Generated] list in the [!UICONTROL Grid Reports] dialog, click ![Delete](/help/search-social-commerce/assets/delete-new.png "Delete") next to the file name.
