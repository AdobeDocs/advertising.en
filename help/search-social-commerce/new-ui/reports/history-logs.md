# (New UI) View the [!UICONTROL History Logs] report

The [!UICONTROL History Logs] report includes a log of changes made to the advertiser account in up the last 31 days. The report can include changes to the following types of objects: users (advertisers), portfolios, campaigns, ad groups, ads, keywords, placements, and product targets. You can sort and filter the data by any column.

You can download additional information about the advertiser's history logs to a file in [!DNL Microsoft Excel] workbook (XLSX file) format. The report includes the the old and new values and the time the change occurred.

>[!NOTE]
>
>For changes to portfolios, see also the [portfolio's change history](/help/search-social-commerce/new-ui/manage/portfolios/portfolio-view-change-history.md).

## View the [!UICONTROL History Logs] report {#history-logs-open}

1. In the main menu, click **[!UICONTROL Reports]** > **[!UICONTROL History Logs]**.

1. (Optional) [Change the data included in the view](/help/search-social-commerce/common-tasks/data-views/data-views-about.md).

## Manage report downloads for [!UICONTROL History Logs]

### Generate a report with the filtered data rows

1. [Open the history logs](#history-logs-open).

1. In the upper right, click ![Download Report](/help/search-social-commerce/assets/download.png "Download Report").

1. In the [!UICONTROL Grid Reports] settings, enter a unique report name, and then click **[!UICONTROL Generate]**.

   By default, the file is named "allChangeHistory_YYYYMMDD_NNNN," where "NNNN" is the sequential job number (such as "allChangeHistory_20260402_1326).

   The file is added to the [!UICONTROL Recently Generated] list.

1. (Optional) To download the file once it's completed, click ![Download](/help/search-social-commerce/assets/download.png "Download") next to the file name.

   The file is downloaded according to your browser's normal procedure.

### Download a completed report

1. [Open the history logs](#history-logs-open).

1. In the upper right, click ![Download Report](/help/search-social-commerce/assets/download.png "Download Report").

1. In the [!UICONTROL Recently Generated] list in the [!UICONTROL Grid Reports] dialog, click ![Download](/help/search-social-commerce/assets/download.png "Download") next to the file name.

   The file is downloaded according to your browser's normal procedure.

### Delete a completed report

1. [Open the history logs](#history-logs-open).

1. In the upper right, click ![Download Report](/help/search-social-commerce/assets/download.png "Download Report").

1. In the [!UICONTROL Recently Generated] list in the [!UICONTROL Grid Reports] dialog, click ![Delete](/help/search-social-commerce/assets/delete-new.png "Delete") next to the file name.
