# Manage postal code lists

You can create and manage lists of postal codes for individual countries for placement targeting. Target or exclude specific postal code lists within placement settings.

## View a postal code list

1. In the main menu, click **[!UICONTROL Resources]** > **[!UICONTROL Postal Code Lists]**.

1. Click the list name.

1. (Optional) To export the selected list to an XLSX ([!DNL Microsoft Excel] spreadsheet) format, click ![Export](/help/dsp/assets/export.png "Export") **[!UICONTROL Export]**.

   The file is downloaded according to your browser's normal procedure.

## Create a postal code list

1. In the main menu, click **[!UICONTROL Resources]** > **[!UICONTROL Postal Code Lists]**.

1. In the upper right, click **[!UICONTROL Create].**

1. Enter the **[!UICONTROL List name]**, select the applicable **[!UICONTROL Country]**, and then click **[!UICONTROL Create].**

1. Do any of the following:

   * To manually enter or paste postal codes to add:

     1. Click **[!UICONTROL Add Postal Codes]**.

     1. Enter or paste up to 25,000 postal codes, with each postal code on a separate line.

     1. Click **[!UICONTROL Validate]** to verify if the postal codes are valid.

        Any invalid postal codes are identified in the [!UICONTROL Validation Results]. Only valid postal codes are added if you continue.

        * To download any invalid postal codes to an XLSX ([!DNL Microsoft Excel] spreadsheet) format, click **[!UICONTROL Download invalid codes]**. The file is downloaded according to your browser's normal procedure.

     1. Click **[!UICONTROL Add to list]**.

   * To remove specific postal codes, do either of the following:

      * To select postal codes to remove:
      
        1. Select the check box next to each postal code to remove from the list.
        
        1. Click **[!UICONTROL Remove]**.
        
        1. In the confirmation message, click **[!UICONTROL Remove]**.

      * To remove all postal code:
        
        1. Click **[!UICONTROL Remove All]**.

        1. In the confirmation message, click **[!UICONTROL Remove All]**.

## Edit a postal code list

1. In the main menu, click **[!UICONTROL Resources]** > **[!UICONTROL Postal Code Lists]**.

1. Click the list name.

1. Do any of the following:

   * To manually enter or paste postal codes to add:

     1. Click **[!UICONTROL Add Postal Codes]**.

     1. Enter or paste up to 25,000 postal codes, with each postal code on a separate line.

     1. Click **[!UICONTROL Validate]** to verify if the postal codes are valid.

        Any invalid postal codes are identified in the [!UICONTROL Validation Results]. Only valid postal codes are added if you continue.

        * To download any invalid postal codes to an XLSX ([!DNL Microsoft Excel] spreadsheet) format, click **[!UICONTROL Download invalid codes]**. The file is downloaded according to your browser's normal procedure.

     1. Click **[!UICONTROL Add to list]**.

   * To remove specific postal codes, do either of the following:

      * To select postal codes to remove:
      
        1. Select the check box next to each postal code to remove from the list.
        
        1. Click **[!UICONTROL Remove]**.
        
        1. In the confirmation message, click **[!UICONTROL Remove]**.

      * To remove all postal code:
        
        1. Click **[!UICONTROL Remove All]**.

        1. In the confirmation message, click **[!UICONTROL Remove All]**.

## Export a postal code list

1. In the main menu, click **[!UICONTROL Resources]** > **[!UICONTROL Postal Code Lists]**.

1. Click the list name.

1. Click ![Export](/help/dsp/assets/export.png "Export") **[!UICONTROL Export]**.

   The file is downloaded in XLSX ([!DNL Microsoft Excel] spreadsheet) format according to your browser's normal procedure.
